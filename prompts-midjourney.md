## Notes d'utilisation :
- **Format** : `--ar 9:16` pour Stories/TikTok, changer en `--ar 16:9` pour bannières web
- **Version** : `--v 6.1` pour le meilleur photoréalisme
- **Style** : `--style raw` pour un rendu plus naturel et moins "IA"
- **Variantes** : Ajouter `--chaos 20` pour plus de diversité dans les résultats
- **Upscale** : Utiliser l'upscale 2x ou 4x après sélection du meilleur résultat
